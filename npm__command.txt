1. Сначала делаем инициализацию через команду npm init и отвечаем на вопросы:

package name - [defender-game"]
version - 1.0.0
description - [Browser based game]
entry point - index.js
test command - оставьте пустым
git repository - [URL вашего GitHub репозитория]
keywords - [game]
author - [ваше имя или псевдоним]
license - ISC

2. Команды для Bable:

npm install --save-dev @babel/core @babel/cli @babel/preset-env
npm install core-js

в package.json в scripts написать "build": "babel src -d dist"

у bable конфигурационный файл создаем сами 


3. Команды для ESlint версии 10>

npm init @eslint/config@latest 

в package.json в scripts написать  "lint": "eslint ."

у ESlint конфигурационный файл создается сам. Мы только добавляем в него игнорирование проверки папки dist и node__module
